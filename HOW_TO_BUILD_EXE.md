# دليل الحصول على ملف exe جاهز — بدون أي CMD

هذا الدليل موجّه لك مباشرة، خطوة بخطوة، بدون أي حاجة لفتح Terminal أو CMD.
كل الخطوات عبر متصفح الإنترنت فقط.

## الخطوة 1 — إنشاء حساب GitHub (لو مش عندك)
اذهب إلى https://github.com/signup وأنشئ حسابًا مجانيًا (بريد إلكتروني + كلمة مرور).

## الخطوة 2 — إنشاء مستودع (Repository) جديد
1. اضغط على "+" أعلى يمين الصفحة ← "New repository"
2. اختر اسمًا (مثلاً: `db-explorer-desktop`)
3. اتركه "Public" (كافٍ لأننا لا نستخدم أي بيانات سرية)
4. اضغط "Create repository"

## الخطوة 3 — رفع ملفات المشروع (سحب وإفلات، بدون Git)
1. في صفحة المستودع الفارغ، اضغط رابط "uploading an existing file"
2. فك ضغط الملف `db-explorer-desktop.zip` على جهازك أولًا
3. اسحب **كل محتويات** المجلد (وليس المجلد نفسه) وأفلتها في صفحة الرفع
   - ملاحظة: GitHub عبر المتصفح أحيانًا لا يرفع المجلدات الفارغة أو الملفات المخفية
     مثل `.github/` بنفس الطريقة — لو حصلت مشكلة، الطريقة الأضمن هي استخدام
     تطبيق **GitHub Desktop** (تطبيق رسمي بواجهة رسومية بالكامل، بدون أي أوامر):
     https://desktop.github.com — تسجّل دخول، تختار "Add local repository"،
     تحدد مجلد المشروع، وتضغط "Publish repository". هذا أسهل وأضمن.
4. بعد الرفع، اكتب أي رسالة في خانة "Commit changes" واضغط "Commit changes"

## الخطوة 4 — مراقبة البناء التلقائي
1. اذهب لتبويب **"Actions"** أعلى صفحة المستودع
2. ستجد عملية بناء باسم "بناء تطبيق سطح المكتب (exe)" تعمل تلقائيًا
   (يبدأ فورًا بعد الرفع، ويستغرق عادة 3-6 دقائق)
3. انتظر حتى تظهر علامة ✓ خضراء بجانبها (يعني نجح البناء)

## الخطوة 5 — تحميل ملف exe
1. اضغط على عملية البناء الناجحة (الاسم اللي عليه ✓ الأخضر)
2. انزل لأسفل الصفحة لقسم **"Artifacts"**
3. اضغط على **"windows-installer"** لتحميله (سيكون ملف zip يحتوي على exe بداخله)
4. فك ضغط الملف الناتج — ستجد ملف
   `مستكشف-قواعد-البيانات-Setup-1.0.0.exe` جاهزًا

## الخطوة 6 — التثبيت على ويندوز
1. انسخ ملف exe لأي جهاز ويندوز (حتى بدون إنترنت بعد كده)
2. شغّله بالضغط المزدوج
3. لو ظهرت رسالة "Windows protected your PC" (SmartScreen):
   اضغط **"More info"** ثم **"Run anyway"** — طبيعي لأن الملف غير موقّع رقميًا
   بشهادة مدفوعة، وهذا لا يؤثر على أمان أو عمل التطبيق
4. اتبع خطوات المثبّت العادية (Next → Next → Install)
5. التطبيق سيعمل بعدها من قائمة ابدأ / سطح المكتب، **بدون إنترنت نهائيًا**

---

## ملاحظة: أي تحديث لاحق للتطبيق
أي مرة تعدّل فيها الكود وترفعه تاني على GitHub (نفس خطوة الرفع)، هيبني exe جديد
تلقائيًا من غير ما تعمل أي حاجة إضافية — فقط ارجع لتبويب Actions وحمّل النسخة الجديدة.
