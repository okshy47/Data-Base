# مستكشف قواعد البيانات

تطبيق ويب تقدمي (PWA) يعمل بالكامل من جهازك — يمكن تثبيته على الكمبيوتر والموبايل
والعمل بدون اتصال بالإنترنت. يسمح باستيراد أي قاعدة بيانات (Excel، CSV، أو SQLite)
والبحث الشامل داخلها، مع الاحتفاظ بها محليًا بشكل دائم.

## الميزات

- **تثبيت كامل offline**: بعد أول فتح، يعمل التطبيق بدون إنترنت نهائيًا (Service Worker)
- **استيراد**: ملفات `.xlsx` / `.xls` / `.csv` / `.sqlite` / `.db`
- **بحث شامل**: بحث نصي عبر كل الجداول، مع فلاتر (قاعدة بيانات / جدول / عمود)
- **عرض وتصفح البيانات**: جدول تفاعلي بصفحات لكل جدول مستورد
- **تصدير**: تصدير جدول كـ CSV أو قاعدة كاملة كملف Excel متعدد الأوراق
- **تخزين دائم**: البيانات تبقى محفوظة على الجهاز حتى بعد إغلاق المتصفح/التطبيق
- **دعم كامل للعربية**: واجهة RTL، وقراءة/كتابة صحيحة للنصوص العربية في كل الصيغ

## التشغيل محليًا

```bash
npm install
npm run dev      # بيئة التطوير
npm run build    # نسخة الإنتاج (dist/) — تتضمن Service Worker جاهزًا
npm run preview  # معاينة نسخة الإنتاج محليًا
```

## التثبيت على جهازك

- **كمبيوتر (Chrome/Edge)**: أيقونة التثبيت ⊕ في شريط العنوان، أو زر "تثبيت" داخل التطبيق
- **أندرويد**: زر "تثبيت" الذي يظهر تلقائيًا داخل التطبيق
- **آيفون/آيباد (Safari)**: زر مشاركة ⬆ ← "إضافة إلى الشاشة الرئيسية" (تعليمات مفصّلة داخل التطبيق)

## ملاحظة عن ملفات Access

ملفات `.mdb`/`.accdb` لا يمكن قراءتها مباشرة داخل المتصفح. صدّرها أولًا من Access كملف
Excel، أو حوّلها إلى SQLite، ثم استوردها هنا.

## البنية التقنية

- Vite + React + TypeScript
- `vite-plugin-pwa` للعمل offline والتثبيت
- `dexie` (IndexedDB) للتخزين المحلي الدائم
- `xlsx` لقراءة/كتابة Excel و CSV
- `sql.js` (WebAssembly) لقراءة قواعد SQLite داخل المتصفح

راجع `PROJECT_STATUS.md` لتفاصيل كل جولة تطوير وما تم إنجازه فيها.
